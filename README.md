# NIKKE_music_py

原作者：https://github.com/Daat-ami/nikke-ttstar

国际服音游助手，台服没做适配

中文说明
这个项目是为了 NIKKE 小游戏自动化测试而创建的。

适配速度：3.0
适配难度：Casual

不能保证所有曲子都能全 Perfect，打个S问题不大

使用方法
1.先使用 DoroHelper 将 NIKKE 窗口调整为 1080p。
2.将 NIKKE 窗口固定在屏幕左上角。（理论上第一步执行完会自动到左上角然后可以不用动了）
3.进入小游戏，并选择 Casual 难度。
4.确认游戏速度为 3.0。
5.启动本项目提供的 exe 程序。
6.启动后，手动将 exe 窗口移开，确保它不要遮挡 NIKKE 游戏画面。
7.如需停止脚本，可按 ESC 退出。

注意事项
本项目目前仍处于测试阶段，仅用于个人学习、实验和脚本测试。
