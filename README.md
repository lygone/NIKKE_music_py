# NIKKE_music_py

原作者：https://github.com/Daat-ami/nikke-ttstar

国际服音游助手，台服没做适配

中文说明
这个项目是为了 NIKKE 小游戏自动化测试而创建的。

当前版本说明
当前适配速度：3.0
当前主要适配难度：Casual
目前并不能保证所有曲子都能全 Perfect
个别曲子可能会出现 Miss、Great 或识别不稳定的情况
使用方法
先使用 DoroHelper 将 NIKKE 窗口调整为 1080p。
将 NIKKE 窗口固定在屏幕左上角。
进入小游戏，并选择 Casual 难度。
确认游戏速度为 3.0。
启动本项目提供的 exe 程序。
启动后，手动将 exe 窗口移开，确保它不要遮挡 NIKKE 游戏画面。
如需停止脚本，可按 ESC 退出。
注意事项
本项目目前仍处于测试阶段，仅用于个人学习、实验和脚本测试。
